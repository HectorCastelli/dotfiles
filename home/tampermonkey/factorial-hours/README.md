# Factorial HR Spanish hours
## Automate the input of shifts in factorial

[greasyfork](https://greasyfork.org/en/scripts/491093-factorial-hr-spanish-hours)

Automates the input of shifts in FactorialHR with the standard spanish time.

This script launches when you open the page for the time input, and will add the hours:

09:00 to 13:00
14:00 to 18:00
The script has a small delay to load (to ensure the page is completely loaded), and will prompt you when it starts and finishes.
